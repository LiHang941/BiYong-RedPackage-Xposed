# Xposed模块-QQSendPoke

### 主要特性
- 解除发送戳一戳次数和时间的限制
- 支持设置戳一戳小功能连发次数
- 支持设置消息发送间隔(秒)

###使用方法
- 安装Xposed框架
- 安装本程序，并在Xposed框架的模块列表勾选，重启手机
 - 若模块依旧失效，在模块列表重新勾选并重启手机
